# Nestter
This is a Twitter clone built with Next.js, TypeScript, and Tailwind CSS.

## Technology Stack
- Next.js + TypeScript
- Tailwind CSS
- [daisyUI](https://daisyui.com/)
- [Drizzle ORM](https://orm.drizzle.team/)
- [Neon](https://neon.tech)
- AWS Lambda, SQS

## Running locally

```bash
bun dev
```
